# Reisen mit Ilja

Landingpage für **reisenmitilija.at** (Next.js, für Vercel geeignet).

## Entwicklung

```bash
npm install
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000).

## Vercel

Repository mit Vercel verbinden, Framework **Next.js** wählen, Root-Verzeichnis ist das Projektroot. Custom Domain `reisenmitilija.at` nach Domain-Kauf in den Vercel-Projekteinstellungen hinterlegen.
